Fullstack build order (single flow)

Step 1 ‚Äì Backend

  1. Express app; PostgreSQL connection.
  2. User model; auth: login ‚Üí JWT.

Tech: Node.js + Express. DB: PostgreSQL. Auth: JWT.

Step 2 ‚Äì Frontend

  1. React app; routing; auth (login, JWT in localStorage, auth context).
  2. Dashboard: fetch summary + list; show monthly total income, expenses, balance.
  3. Add income/expense form (amount, date, category, note).
  4. Expense list with edit/delete; filter by date range and category.
  5. Charts: e.g. spending by category, monthly trend (Chart.js or Recharts) using API data.
Tech: React. Charts: Chart.js or Recharts.

Step 3 ‚Äì Integration
