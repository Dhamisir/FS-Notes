🤔 Conditionals (If / Else)

Conditionals allow our code to make decisions. "If this happens, do that; otherwise, do this."

Basic If Statement:

let hour = 14;

if (hour < 12) {
  console.log("Good Morning!");
} else {
  console.log("Good afternoon!");
}

Else If (Multiple Choices):

let score = 85;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else {
  console.log("Grade: C");
}
