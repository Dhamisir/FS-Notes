🎯 Day 1 Final Assignments

Complete these tasks in a single assignment.js file and link it to an index.html file.

Task 1: The Bio Display

  • Create variables for firstName, lastName, age, and isLearner.
  • Use JS to combine names into a fullName variable.
  • Display this info on the web page using document.body.innerHTML.

Task 2: Simple Calculator

  • Take two numbers: num1 = 20, num2 = 10.
  • Print their Sum, Difference, Product, and Quotient in the console.

Task 3: Voting Eligibility

  • Create an age variable.
  • If age >= 18, log "You are eligible to vote!"
  • Otherwise, log "Too young to vote."

Task 4: Table Generator

  • Write a for loop to print the table of 2 (2, 4, 6... 20).
  • Bonus: Try printing it on the HTML page using document.write().
