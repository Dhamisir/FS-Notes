🛠 Express.js Setup & Your First API

Express is a fast and minimal framework for Node.js. It simplifies HTTP handling and routing.

1. Installation

npm install express

2. Your First Server

const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
  res.send('Hello Backend World!');
});

app.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}`);
});

3. API endpoint for Testing (/user)

Let's create an endpoint that returns data in JSON format.

app.get('/user', (req, res) => {
  res.json({
    id: 1,
    name: "John Doe",
    role: "Developer"
  });
});
