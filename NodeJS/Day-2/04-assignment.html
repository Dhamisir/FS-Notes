🎯 Day 2 Project: The User Management API

Combine Express, Folder Structure, and Logic.

Task 1: Basic Server Setup

  • Initialize npm init -y.
  • Install express.
  • Create index.js as the main file.

Task 2: API Architecture (MVC)

  • Create routes/user.routes.js.
  • Create controllers/user.controller.js.
  • Build a /profile route that returns a hardcoded user object.

Task 3: Multiple Routes

  • Add a POST /login endpoint (even if it just returns "Logged in").
  • Add a GET /all endpoint to list users.
  • Test all routes using Postman or your browser.

Task 4: Bonus Challenge

  • Add a simple middleware that logs the current time on every request.
