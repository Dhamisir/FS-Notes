⚙️ The Node.js Runtime

Node.js is not a programming language; it’s a runtime environment that lets you execute JS code externally from your browser.

1. Checking Node Version

node -v

2. Running Your First File

  1. Create index.js
  2. Write console.log("Hello Node") inside it.
  3. In Terminal, type: node index.js

3. The Global Object

Instead of window, we have global.

console.log(global);
