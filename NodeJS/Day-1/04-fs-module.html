📁 File System (fs Module)

JavaScript in the browser cannot read files from your computer. But Node.js can! We use the built-in fs module.

1. Writing a File

const fs = require('fs');

fs.writeFileSync('hello.txt', 'Hello from Node.js!');
console.log("File created!");

2. Reading a File

const fs = require('fs');

const data = fs.readFileSync('hello.txt', 'utf-8');
console.log(data); // Hello from Node.js!

3. Appending & Deleting

// Add more text
fs.appendFileSync('hello.txt', '\nMore data here.');

// Delete file
// fs.unlinkSync('hello.txt');
