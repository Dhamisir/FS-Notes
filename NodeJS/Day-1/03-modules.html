📦 Node.js Modules

Node treats every file as a "Module". We can export properties from one file and import them into another.

1. Exporting

// helpers.js
function add(a, b) { return a + b; }

module.exports = add; // This exports the function

2. Importing (require)

// index.js
const addFn = require('./helpers.js');

console.log(addFn(10, 20)); // 30

3. Core Modules

Node has built-in modules like path, os, and http.

const os = require('os');
console.log(os.platform()); // 'win32' or 'linux'
