🎯 Day 1 Graduation Assignment: Node Essentials

Execute these tasks using your terminal.

Task 1: The Calculator Module

  • Create math.js with add, sub, and mul functions.
  • Export them using module.exports.
  • Import them into app.js and show some results.

Task 2: File Logger

  • Write a script that creates a log file activity.txt.
  • Use fs.appendFile to add a timestamp of when the script was run.

Task 3: OS info

  • Use the os module to find your computer's "Free RAM" and "Home Directory".
