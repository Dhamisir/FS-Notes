♻️ JS Basics (Recap for Backend)

Before jumping into Node, ensure you understand how JS works on the terminal without a browser.

1. No DOM on Server

There is no document or window in Node.js.

// Browser JS:
document.getElementById("my-id").innerHTML = "Hello"; // Works

// Node.js (Server) JS:
document.getElementById("my-id").innerHTML = "Hello"; // ERROR!

2. Printing to Terminal

Use console.log() to see output in your command prompt / terminal.

console.log("Server side is running...");

3. ES6 Features (Arrow Functions)

Node fully supports modern JS syntax.

const add = (a, b) => a + b;
console.log(add(5, 5)); // 10
