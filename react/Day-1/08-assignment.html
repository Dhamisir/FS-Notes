Task 4 (Theoretical) The "Why React" Summary

Write down 3 differences between a Traditional Website (MPA) and a Single Page Application (SPA) in your own words.

üí° View Solution
  • Reloading: MPA reloads the whole page on every click; SPA only updates the part that changed.
  • Performance: SPA feels faster and smoother after initial load; MPA can have "white flashes" between pages.
  • Data: MPA requests full HTML from server; SPA usually requests only data (JSON) and updates UI using JavaScript.

üöÄ Why React is Powerful?

Imagine building a Facebook feed. In traditional JS, adding a new post means manually finding the container, creating 10+ elements (div, img, span, p), setting their classes, and appending them. In React, you just add the post to a list (Array), and React handles the UI!
