üß† What are Props?

Think of props like arguments passed to a function. They allow you to pass data from a Parent component to a Child component.

üß© Simple Example

Imagine a Student ID Card component. The "machine" (component) is the same, but the student's name changes.

Defining the Component (Child)

function StudentCard(props) {
  return (
    <div>
      <h2>Name: {props.name}</h2>
      <p>Roll No: {props.roll}</p>
    </div>
  );
}

Using the Component (Parent)

function App() {
  return (
    <div>
      <StudentCard name="Rahul" roll="101" />
      <StudentCard name="Anita" roll="102" />
    </div>
  );
}

A component can never modify its own props. If you want to change data, you need State (our next topic).
