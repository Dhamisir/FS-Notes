React doesn't have a built-in router, so we use a very popular library called react-router-dom.

Step 1

Install the Library

npm install react-router-dom
Step 2

Wrap your App with BrowserRouter

In your main.jsx or App.jsx, import BrowserRouter and wrap your entire application code with it.

import { BrowserRouter } from "react-router-dom";

function App() {
  return (
    <BrowserRouter>
      <RoutesHere />
    </BrowserRouter>
  );
}
üìå BrowserRouter: It watches the URL in the browser address bar and tells React when it changes.
Step 3

Define your Routes

Use Routes as a container and Route to map a URL to a component.

import { Routes, Route } from "react-router-dom";

function RoutesHere() {
  return (
    <Routes>
      <Route path="/" element={<Home />} />
      <Route path="/login" element={<Login />} />
      <Route path="/about" element={<About />} />
    </Routes>
  );
}
