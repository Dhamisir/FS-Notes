What if you have 1,000 users? You can't create 1,000 routes like /user/1, /user/2, etc. Instead, we use Dynamic Parameters.

1Ô∏è‚É£ Setting up a Dynamic Route

Use a colon : followed by a name (like id or username) to mark a part of the URL as dynamic.

<Route path="/user/:id" element={<UserProfile />} />

2Ô∏è‚É£ Reading the Parameter (useParams)

To use that id inside your component, React Router provides a hook called useParams.

import { useParams } from "react-router-dom";

function UserProfile() {
  const { id } = useParams();

  return (
    <div>
      <h1>User Profile ID: {id}</h1>
    </div>
  );
}
