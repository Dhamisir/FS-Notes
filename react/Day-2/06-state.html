What is State?

State is data that lives inside a component and can change over time. When state changes, React automatically updates the UI.

üß© The Counter Example

We use the useState hook to create state variables.

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Current Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increase
      </button>
    </div>
  );
}

üîç Breaking it Down

  • count: The current value (starts at 0).
  • setCount: The function used to update the value.
  • useState(0): Tells React to create a piece of memory with 0 as the initial value.

‚ùå NEVER Update State Directly!

Do not do: count = count + 1;

Always do: setCount(count + 1);

Reason: React only knows it needs to update the UI when you call the "setter" function (setCount).
