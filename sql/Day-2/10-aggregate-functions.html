Function Use Example

1. COUNT() - Count Rows
üëâ Meaning
Returns the number of rows
üß† Example Code

  • Total number of users

üß† Example Code

‚úÖ Aggregate Functions with GROUP BY
üß† Example Code

üí° Note: When using GROUP BY, aggregate functions calculate per group
‚úÖ Complete Example 
-- Statistics per city (only active students)
SELECT 
    city,
    COUNT(*) as total,
    AVG(age) as avg_age,
    MAX(age) as oldest,
    MIN(age) as youngest
FROM students 
WHERE status = 'active'
GROUP BY city
HAVING COUNT(*) > 1
ORDER BY total DESC;
