Symbol Meaning Example

1. Names Starting With...
üëâ Pattern: 'A%'
Matches any string starting with 'A'
üß† Example Code

üìù Result: Shows names like Amit, Anjali, Aman

üëâ Pattern: '%a'

üß† Example Code

üìù Result: Shows names like Neha, Priya, Anjali

üëâ Pattern: '%ra%'

üß† Example Code

üëâ Pattern: 'A_'
