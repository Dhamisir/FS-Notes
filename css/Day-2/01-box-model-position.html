A. The "Border-Box" Revolution

By default, browser uses content-box. This means if you set a width of 200px and add 20px padding, the box becomes 240px. This is a nightmare for layouts.

/* THE SOLUTION: Add this to the top of every CSS project */ * { box-sizing: border-box; margin: 0; padding: 0; }
Why? With border-box, width: 200px means the box stays 200px, and padding/border grow inward.

2. Advanced Positioning

Positioning allows you to take elements out of the normal "document flow".

A. The Position Family

  1. Static: Default. Elements follow each other naturally.
  2. Relative: Moves slightly from its original spot without affecting neighbors. Crucial as a "parent" for Absolute.
  3. Absolute: Teleports the element relative to its nearest "Relative" parent.
  4. Fixed: Glued to the viewport. Stays there even when you scroll.
  5. Sticky: A hybrid. Behaves normally until it reaches a scroll point, then sticks.

B. Z-Index (The Third Dimension)

Controls which element is "on top". Only works on positioned elements (relative, absolute, etc.).

z: 1
z: 10
z: 5
.modal { position: fixed; z-index: 999; /* Always on top of everything */ }
