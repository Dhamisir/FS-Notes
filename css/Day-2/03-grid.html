Flexbox is for one-way layouts. Grid is for two-way (Rows AND Columns) layouts. Let's build it piece by piece.

Step 1: The Skeleton

We start with a parent and children. Initially, they just stack.

<div class="container"> <div class="item">1</div> <div class="item">2</div> <div class="item">3</div> </div>

Step 2: Create the Columns

Tell the container how many columns you want using grid-template-columns.

.container { display: grid; grid-template-columns: 200px 200px 200px; /* 3 Columns of 200px each */ }

Observation: The boxes are side-by-side, but they are "hard coded" to 200px. What happens if the screen gets smaller? They will break or cause scroll.

Step 3: Use Flexible Units (fr)

fr stands for Fraction. It means "take 1 share of whatever space is left".

.container { display: grid; grid-template-columns: 1fr 2fr 1fr; } /* OR shortcut for 3 equal columns */ grid-template-columns: repeat(3, 1fr);

Step 4: The Clean Space (Gap)

Stop using margin to space out boxes. Use gap.

.container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; /* Perfectly even space between all rows and columns */ }

Step 5: Spanning (Merging Cells)

You can tell an item to take up multiple cells. This is how you make Sidebars and Main content.

.item-1 { grid-column: span 2; /* Take 2 columns */ } /* Specific Line Spanning */ .header { grid-column: 1 / 4; /* Start at line 1, end at line 4 (Full width) */ }

Step 6: The "Holy Grail" (Auto-fit & Minmax)

This is the professional way to build a grid that works on all screens without media queries.

.container { display: grid; /* Magically wraps boxes as the screen grows/shrinks */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; }
Breakdown:
- auto-fit: Fit as many as you can.
- minmax(250px, 1fr): Every box must be at least 250px, but can grow to fill the rest.

This is the standard way to change layout for Desktop, Tablet, and Mobile.

/* 1. Turns on grid and sets 3 columns for Desktop */ .container { display: grid; gap: 20px; grid-template-columns: repeat(3, 1fr); } /* 2. Tablet = 2 columns (below 900px) */ @media (max-width: 900px) { .container { grid-template-columns: repeat(2, 1fr); } } /* 3. Mobile = 1 column (below 600px) */ @media (max-width: 600px) { .container { grid-template-columns: 1fr; } }

üéì How to Explain to Students:

  1. display: grid ‚Üí turns on the grid engine.
  2. grid-template-columns ‚Üí controls the number and size of columns.
  3. repeat(3, 1fr) ‚Üí creates 3 equal fractions of space.
  4. Media query ‚Üí changes columns for different screen widths.

Standard Breakpoints:
üì± Mobile: 0px to 600px (1 column)
üì± Tablet: 600px to 900px (2 columns)
üíª Desktop: 900px and above (3 columns)
