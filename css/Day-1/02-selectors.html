Selectors are used to "find" (or select) the HTML elements you want to style. There are 3 main types used by beginners:

1Ô∏è‚É£ Element Selector

Targets all elements with a specific tag name.

/* Styles ALL h1 tags on the page */
h1 {
  color: blue;
}

Use case: Setting default styles for the whole site (e.g., body font, paragraph spacing).

2Ô∏è‚É£ Class Selector (.)

Targets elements with a specific class attribute. It starts with a dot.

/* HTML */
<button class="btn-primary">Click Me</button>
<p class="btn-primary">I am just text with class</p>

/* CSS */
.btn-primary {
  background-color: blue;
  color: white;
}

Gold Rule: Classes are **Reusable**. You can use the same class on many elements.

3Ô∏è‚É£ ID Selector (#)

Targets a specific element with a unique id. It starts with a hash.

/* HTML */
<div id="main-header">Logo Here</div>

/* CSS */
#main-header {
  height: 100px;
  background-color: #333;
}

Gold Rule: ID must be **Unique**. One ID per page. IDs have higher priority than classes.
